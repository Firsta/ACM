    被其中一个人拿过的牌，另一个人肯定是拿不了了,所以可以先枚举总共有多少取法，A为男生取的牌集合，B为女生取的牌集合，F(A)、F(B)为各自牌上数字的异或和。枚举A交B=空集的取法可以用循环:
	for(int j=temp;;j=(j-1)&temp)
	{
		cnt++;
		if(j==0)
			break;
	}
    总的取法中F(A)>F(B)的取法等于F(B)>F(A)的取法，把两人的牌交换可得，因为F(A)=F(B)也算女生赢，总和=|F(A)>F(B)|+|F(A)<F(B)|+|F(A)=F(B)|=2*|F(A)<F(B)|+|F(A)=F(B)|，得F(A)<=F(B)=(总和+|F(A)=F(B)|)/2。
    要算|F(A)=F(B)|，即|F(A)^F(B)|=0。
